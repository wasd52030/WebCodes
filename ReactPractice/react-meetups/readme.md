# React Meetups
這玩意是在清明連假在ITHelp刷的React文中的實踐範例，透過這次的閱讀，讓我對React有進一步的了解，特別是對Context這個觀念。

# 技術選型
使用React+Typescript，構建工具使用Vite。

# 專案使用
git clone
```git clone ```
進入專案目錄
```cd react-meetups```
安裝依賴項目
```npm install```
啟動專案
```npm run dev```
